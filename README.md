# ReelGuard

🎬 **Know where your time goes.**

ReelGuard is a premium Android application designed to help users track and control their Instagram Reels consumption. Using advanced Accessibility Services, it provides real-time analytics, daily goals, and creator-specific insights in a sophisticated, glassmorphism-inspired UI.

## Features
- **Precise Reel Counting:** Structural detection of reel transitions using scroll events.
- **Creator Analytics:** Identify which creators you watch most frequently.
- **Productivity Goals:** Set and monitor daily reel limits.
- **Weekly Engagement:** Visualize your habits with animated trend charts.
- **Privacy First:** All tracking data is stored locally on your device using Room database.
- **Premium UI:** Modern dark theme with smooth animations and glassmorphism effects.
